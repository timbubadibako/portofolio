# Architecture Overview
status: ACTIVE
version: 1.0.4

## System Vision
The DevPortal is architected as a dual-mode interface (TUI/GUI) bridging the gap between low-level system interaction and high-fidelity visual storytelling.

### Core Stack
- **Next.js 16** (App Router)
- **Tailwind CSS v4** (Industrial Theme)
- **Framer Motion** (Physical Transitions)
- **GSAP** (Scroll Orchestration)

### Security Layer
All transmissions are filtered through the CRT phosphor overlay to ensure 100% aesthetic integrity.
